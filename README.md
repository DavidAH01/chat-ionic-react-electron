# Chat (Ionic - React - Electron)

## Setup
- npm install
- Rename .env.example to .env
- Config firebase's keys on .env file
- npm run electron-dev or npm run electron-pack (prod)

## Frameworks and development libraries
- Ionic V5
- React
- Electron
- Axios
- Firebase
- Easy peasy (State manager)

## Features
- One to one chat
- Delete messages
